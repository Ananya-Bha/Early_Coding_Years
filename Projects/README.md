
# Projects

### Overview

This folder contains a collection of my early coding projects created during my first years of learning programming. These projects were made using Scratch and LiveCode, and they show my progression from block-based programming to simple text-based coding.

The projects in this folder focus on creativity, logic, basic game design, user interaction, and beginner-level problem-solving.

## Projects Included

| Project | Tool | Description |
|---|---|---|
| **A Winter Adventure** | LiveCode | An interactive winter-themed project that helped me practise basic LiveCode commands and user interaction. In this game, users were asked to solve multiple Christmas-based riddles to succeed. |
| **Adventure Escape Room** | LiveCode | A simple escape-room style project focused on decision-making, navigation, and interactive storytelling. |
| **Caesar Cipher** | LiveCode | A beginner encryption project that introduced me to text manipulation and simple cipher logic. |
| **Pig Latin Translator** | LiveCode | A language-based project that helped me practise string handling and rule-based transformations. |
| **Clowns VS Zombies** | Scratch | A game-style Scratch project focused on characters, events, movement, and simple gameplay logic. In this game, users were asked to shoot incoming zombies to succeed and survive. |
| **Escape The Abandoned Ship** | Scratch | An adventure-themed project using Scratch blocks to create interaction, story progression, and challenges. It involves navigating through different rooms in an abandoned ship with obstacles to make it out alive. |
| **Find Your Way Back** | Scratch | A navigation-based Scratch project that helped me practise movement, sequencing, and conditional logic. |
| **Play it To Win It** | Scratch | A Scratch game project focused on basic game mechanics, interaction, and win/lose conditions. It is a Scratch Trailer for an upcoming movie. |
| **The Might Terror** | Scratch | A creative Scratch project using characters, events, and visual storytelling. |
| **The Mighty Halloween Scare** | Scratch | A Halloween-themed Scratch project that combined animation, interaction, and simple game design. |

### What I Learned

Through these projects, I learned how to turn simple ideas into interactive programs. I developed a stronger understanding of sequencing, conditions, events, and basic user interaction. These projects also helped me build confidence in experimenting, debugging, and improving my work over time.

### Reflection
This folder represents an important early stage in my coding journey. While the projects are simple compared to my later work, they helped me build the foundation for programming logic, creativity, and problem-solving. They also show how I gradually moved from visual programming in Scratch to writing more structured code in LiveCode.
